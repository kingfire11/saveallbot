# SaveAllBot

Downloads photos and videos from 1000+ sites (YouTube, Instagram, TikTok, Twitter, Pinterest, etc.) via yt-dlp and sends them back in Telegram.

## Setup

```bash
pip install -r requirements.txt
python bot.py
```

## Features

- Paste any URL → bot downloads and sends the file
- Quality picker for multi-quality videos (360p / 480p / 720p / 1080p / best)
- 50 MB file size limit with clear error message
- Russian / English interface — pick on /start
- Temp files cleaned up after each download

## Commands

- `/start` — language selection
